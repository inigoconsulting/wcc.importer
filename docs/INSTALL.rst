wcc.importer Installation
-------------------------

To install wcc.importer using zc.buildout and the plone.recipe.zope2instance
recipe to manage your project, you can do this:

* Add ``wcc.importer`` to the list of eggs to install, e.g.:

    [buildout]
    ...
    eggs =
        ...
        wcc.importer

* Re-run buildout, e.g. with:

    $ ./bin/buildout
